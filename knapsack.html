knapsack Problem

El problema de la mochila es un desafío de optimización que consiste en seleccionar los objetos con mayor valor posible para incluir en una mochila con capacidad limitada, respetando la restricción de peso. Cada objeto tiene un peso y un valor asignado, y el objetivo es encontrar la combinación óptima que maximiza el valor total.

Se puede resolver con diferentes métodos, como programación dinámica, algoritmos voraces y búsqueda, cada uno con sus características particulares, pero nos vamos a enfocar especialmente en los algoritmos voraces, ya que con este se pueden resolver tomando como entrada diferentes parametros.

Existen variantes como el Problema de Mochila 0/1 (donde se toma el objeto completamente o no) y el Problema de Mochila Fraccional (donde se pueden tomar fracciones de un objeto).
